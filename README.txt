# battlescape
A project to create a starship, turn based with real time graphics game.
Created in Unity by Ariel Talmi, with the help and guidance of Yali Zak-Malkin.


To play the last version of the game, run the file game.exe in the bin directory.

Controls: 
  Right click for movement.
  Cancel movement by right clicking on a faded ship.
  Left click to choose a ship.
  Left clicking on faded ship opens a menu to give commands.
