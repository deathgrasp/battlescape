# battlescape
A project to create a starship, turn based with real time graphics game.
Created in Unity by Ariel Talmi, with the help and guidance of Yali Zak-Malkin.


To play the last version of the game before the project was discontinued, run the file game.exe in the bin directory.

controls: right click for movement, cancelling movement by right clicking on a faded ship. left click to choose a ship, clicking on faded ship opens a menu to give commands.
